# Hough Cercle Detection

## Makefile command line
### Build
Build the program
```Makefile
make build
```

### Run 
Run the program
```Makefile
make run IMG_PATH=image_path
```

### Clean 
Clean the ```bin/``` directory
```Makefile
make clean
```

### Docs
Transfrom the TP CR markdown into LaTeX pdf file. 
```Makefile
make docs
```

### Py 
Run python sample program
```Makefile
make py
```
